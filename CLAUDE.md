# LoooX Configurator — Claude Context

## WAT Framework structuur

Dit project volgt het WAT-framework:

```
workflows/    → markdown SOP's (wat te doen, hoe, welke tools)
tools/        → uitvoerbare scripts (deterministische uitvoering)
.tmp/         → tijdelijke bestanden (wegooibaar)
.env          → API keys (nooit committen)
```

**Werkwijze:** Lees altijd eerst de relevante workflow in `workflows/` voor je iets bouwt of uitvoert. Check `tools/` of er al iets bestaat.
